# Portfolio Web Application made with ReactJS

A personal portfolio web application built with ReactJS to showcase my projects, skills, and experiences. This app highlights my work as a frontend developer and content creator. The app is designed to be responsive, user-friendly, and includes interactive features like viewing, saving, and sharing blog posts.

## Features

- Display of personal projects with links and descriptions.
- Responsive layout for viewing on desktop and mobile devices.
- Links to active blog posts and an easy way to browse them.
- Interactive elements for users to favorite and save blog posts.
  
## Technologies Used

- **ReactJS**: Used for building the UI components and handling the state management.
- **CSS** (or any styling framework you used): For styling and layout.
- **React Router** (if applicable): For navigation between pages.
- **Other libraries/tools**: (e.g., Axios for API calls, Material UI for design components, etc.)
